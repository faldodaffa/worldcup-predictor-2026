⚙️ Tahap 1 — Monte Carlo Engine (3000 iterasi)

Setiap turnamen disimulasikan 3000 kali secara acak (berjalan asinkronus di belakang layar). Proporsi kemenangan di 3000 skenario inilah yang melahirkan persentase absolut di Global Ranking.

🌍 Tahap 2 — Elo Rating Dinamis

Kekuatan awal tim bertumpu pada Elo Rating. Selisih poin Elo antara dua tim menentukan rasio Bivariate Poisson Lambda (λ) untuk pengacakan ekspektasi gol (xG).

🏠 Tahap 3 — Modifikator Geografis (Stadion)

Tuan rumah mendapat buff moral (+0.15 xG). Stadion spesifik memberikan debuff pada lawan: Azteca (ketinggian), Miami (kelembapan ekstrim), Houston (suhu indoor dengan AC).

⚡ Tahap 4 — Momentum Berantai & Underdog Rally

Tim kecil yang berhasil menumbangkan Raksasa secara mengejutkan akan mendapat Momentum Boost permanen. Ada juga faktor Underdog Rally untuk negara spesifik yang memiliki tren lonjakan performa historis.

🎲 Tahap 5 — Threshold: Smart Upsets (Power Gap ≤ 25%)

Di dalam bagan Knockout, jika selisih probabilitas (Gap) antar tim > 25%, Raksasa dipastikan menang secara matematis. Namun jika Gap ≤ 25% (laga sengit / badai momentum), mesin mengaktifkan mode Smart Upsets: Dadu Poisson dilempar murni, mengizinkan lahirnya Kuda Hitam seperti Maroko 2022!

🏆 Tahap 6 — Overview Follows Bracket

Dasbor Overview (Juara, Finalis, Kuda Hitam, Flop) Bukanlah sekadar tebakan dari rata-rata Monte Carlo, melainkan diekstrak secara mutlak dari apa yang divisualisasikan oleh Bagan Knockout. Ini menjamin konsistensi 100%.

📈 Tahap 7 — Sinkronisasi API Real-Time

Node.js server memantau worldcup26.ir setiap 30 detik. Perubahan skor riil akan men-trigger ulang seluruh infrastruktur Python di atas untuk merajut realitas probabilistik baru.

Aplikasi ini adalah proyek riset personal. Semua prediksi bersifat probabilistik dan tidak menjamin hasil aktual pertandingan.
